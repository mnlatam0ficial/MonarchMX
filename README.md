# Monarch Mx — Premium Sneaker Catalog

E-commerce showroom minimalista optimizado para dispositivos móviles y entornos web de alta velocidad. Desarrollado con una arquitectura desacoplada sin dependencias externas (Vanilla JS / CSS3 nativo).

## Optimizaciones de Arquitectura
- **Performance de Filtros:** Motor de búsqueda predictivo capaz de cruzar cadenas de texto (SKU, marca, modelo) y aplicar ordenamiento matemático de precios en tiempo real sin repintar el DOM innecesariamente.
- **Enrutamiento Limpio:** Integración nativa con la API de mensajería para automatizar el embudo de ventas, generando textos estructurados específicos según el producto consultado.
- **Diseño Aristocrático:** Paleta de colores fríos basada en una escala cromática de platino y negros de alta profundidad para potenciar el valor percibido del producto final.

## Despliegue en Servidores
El proyecto está listo para ser alojado de manera estática y gratuita en plataformas globales como **GitHub Pages**, Vercel o Netlify con cero configuraciones de backend adicionales.
